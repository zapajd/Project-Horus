# Project Horus

## Overview

Project Horus is a cybersecurity portfolio initiative focused on designing, securing, monitoring, and documenting enterprise infrastructure using industry best practices.

Inspired by Horus, the ancient symbol of protection, vigilance, and kingship, this project demonstrates the implementation of modern security controls across identity, endpoint, vulnerability management, monitoring, and incident response domains.

The goal is to simulate real-world enterprise security projects while producing professional documentation, architecture designs, operational procedures, and technical implementations.

---

## Organization

### Horus Technologies

Horus Technologies is a fictional enterprise organization used throughout this project to simulate real-world business requirements, security challenges, and infrastructure deployments.

Industry: Technology Services

Employees: 250

Locations: Toronto Headquarters with remote workforce support

Security Objectives:

* Protect organizational identities
* Reduce attack surface
* Implement least privilege access
* Improve visibility and monitoring
* Establish vulnerability management processes
* Create repeatable operational procedures

## Program Phases

### Phase 1 – Active Directory Security Modernization

Design, deploy, secure, and document a modern Active Directory environment.

Key Deliverables:

* Active Directory Domain Services
* DNS Infrastructure
* Organizational Unit Design
* Role-Based Access Control (RBAC)
* Administrative Tiering
* Group Policy Management
* Endpoint Hardening
* Microsoft Defender Configuration
* Vulnerability Management Integration
* Monitoring and Logging
